## Description
I am going to create a top down shooter game with spawning enemies. This is relevant to media and digital arts because there will be a custom made map, sound fx, and animations.

## Features
- Collision
	- Get the location of where the player collides with an object and then place the player next to the object instead of overlapping it.
- Enemy Spawn
	- Create a timer to spawn instances of enemies after a certain amount of time.
- Character Animation
	- Import images of character movement and 
- Shooting
    - Get the location of the gun and update the end of the gun with projectile.

## Challenges
- Learn how to make a map using the program Tiler.
- Learn how to use sprites for collision.
- Learn how to move the camera with the player.
